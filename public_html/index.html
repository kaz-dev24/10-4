Bcrypt for Password Hashing & Salting

To ensure user password security, we utilize bcrypt for password hashing and salting. Bcrypt generates a cryptographic hash that mitigates brute-force attacks and password cracking.


const bcrypt = require('bcrypt');
const saltRounds = 10;

// Hashing a password before saving
bcrypt.hash(password, saltRounds, (err, hash) => {
    if (err) throw err;
    console.log("Hashed Password:", hash);
});

Password Strength Validation

We use a JavaScript function (pwd_validator.js) to evaluate password strength in real-time based on length, uppercase letters, numbers, and symbols.


function evaluatePasswordStrength(password) {
    let strength = 0;
    if (password.length > 10) strength++;
    if (/[A-Z]/.test(password)) strength++;
    if (/\d/.test(password)) strength++;
    if (/[^A-Za-z0-9]/.test(password)) strength++;
    return strength;
}
